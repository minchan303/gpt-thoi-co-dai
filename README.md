This project is a deploy-ready Node.js + static frontend project optimized for Render.
It includes an improved, modern UI (header, cards, buttons), full client JS, and backend.

Quick start (local):
1. copy .env.example -> .env and set OPENAI_API_KEY
2. npm install
3. npm start
4. Open http://localhost:3000

Deploy on Render:
- Push this repo to GitHub
- Create a new Web Service on Render, connect repo
- Build command: npm install
- Start command: npm start
- Add environment variables on Render (OPENAI_API_KEY)
